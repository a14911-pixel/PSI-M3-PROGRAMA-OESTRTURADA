# PSI-M3-PROGRAMA-OESTRTURADA
# 📘 Exercícios de Python – Módulo 3

Este repositório contém os exercícios desenvolvidos em **Python**, no âmbito do **Módulo 3 da disciplina Programação de Sistemas Informáticos**, do **Curso Profissional Técnico de Gestão e Programação de Sistemas Informáticos (GPSI)**.

Os exercícios abordam conceitos fundamentais da programação, desde instruções básicas até estruturas mais avançadas como ciclos, listas e funções.

---

## 🧠 Conteúdos Abordados

Ao longo deste módulo foram trabalhados os seguintes tópicos:

- ✅ Instruções básicas (`print`, variáveis, comentários)
- ✅ Entrada e saída de dados (`input`)
- ✅ Operadores:
  - Aritméticos
  - Relacionais
  - Lógicos
- ✅ Estruturas de decisão:
  - `if`
  - `elif`
  - `else`
- ✅ Ciclos:
  - `while`
  - `for`
- ✅ Listas e operações associadas
- ✅ Funções
- ✅ Programas com menus
- ✅ Resolução de problemas práticos

---

## 📝 Organização dos Exercícios

### 📌 Exercícios Práticos Iniciais
- Impressão de texto no ecrã
- Utilização de variáveis
- Cálculo da idade
- Soma de dois números
- Conversão de temperaturas (Celsius ↔ Fahrenheit)
- Operações matemáticas básicas

---

### 📌 Estruturas de Decisão (IF / ELSE)
- Verificação de saldo bancário
- Classificação por idades
- Número maior entre vários valores
- Par ou ímpar
- Validação de utilizador e palavra-passe
- Avaliação de notas escolares
- Verificação de intervalos
- Ano bissexto

---

### 📌 Ciclos (WHILE e FOR)
- Contagem crescente e decrescente
- Números pares
- Soma de números
- Tabuada
- Cálculo de fatorial
- Menus interativos com repetição

---

### 📌 Listas
- Adicionar notas
- Listar notas
- Calcular média
- Filtrar notas positivas
- Menu de gestão de notas

---

### 📌 Funções e Projeto Final
- Definição e utilização de funções
- Sistema de **Gestão de Stock**, incluindo:
  - Registo de materiais
  - Consulta de stock
  - Atualização de quantidades
  - Visualização geral do stock
- Programa estruturado com menu principal
